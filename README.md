# mt5_python

MT5 Python API

## Prerequisites

1. Python 3.9
2. `virtualenv` package

## Installation

1. Install virtual environment:
    ```bash
    python3 -m venv .venv
    ```

2. Activate virtual environment:
    ```bash
    source .venv/bin/activate
    ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
